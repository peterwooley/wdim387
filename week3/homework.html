WDIM87 Advanced Scripting

Homework: Drawing on Canvas, due Week 4.

Please turn in your homework by emailing me your GitHub Repository URL and Live URL.

Requirements

  1. Implement a drawing app in Canvas.
  2. Allow the user to click and drag on the canvas to draw lines.
  3. Lines should only be drawn where the user is actively clicking the mouse.
  4. Implement a Clear button that removes all existing lines from the canvas (In the canvas, or just in the DOM).
  5. Implement a way for users to change the color of the line.
  6. For 10% Extra Credit: Investigate toDataUrl() and allow the user to move a snapshot of their drawing to an <img /> tag on the same page.
