$.getJSON("http://api.flickr.com/services/feeds/photos_public.gne?format=json&jsoncallback=?",function(data) {
	alert(data.items[0].media.m);
});

JSON — JavaScript Object Notation

JSON is a lightweight data-interchange format.

For us, it gets data from the server to the client and back again.

It is valid JavaScript.

It's simple for Server-side languages to parse and even simpler for JavaScript.

Here's a delicious sample:

{"menu": {
  "id": "file",
  "value": "File",
  "popup": {
    "menuitem": [
      {"value": "New", "onclick": "CreateNewDoc()"},
      {"value": "Open", "onclick": "OpenDoc()"},
      {"value": "Close", "onclick": "CloseDoc()"}
    ]
  }
}}
