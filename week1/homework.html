Homework: First Project

Please turn in your files in a .zip file, using the following naming convention: lastnameFirstname_firstProject.zip

First Project; due Week 4.

Abstract

You will have three (3) weeks to design, develop, and deploy an Ajax-driven Content Management System.

You may work on the project alone, in pairs, groups of three, groups of four, or a group of five.

Requirements

  • The content management system must allow the user to modify one (1) page.
  • This page must allow for inline editing. (e.g., when the user wishes to modify the content, they can edit and save the changes without reloading the page.)
  • The page must consist of, at least, three editable areas: the header, the main content, and the footer.
  • The interaction design must be just that, designed.
  • Generated JavaScript Documentation must accompany the source code that is turned in.
  • The application must be hosted on a server that can be accessed from the Internet.

Unrequirements

  • A login system is not necessary, but even a basic system (implemented well) will grant 10% extra credit.

Grading Rubric

  • 20% Interaction Design
  • 30% Code functions properly, is written clearly, has good comments.
  • 50% Completed the scope of the project, project is of sufficient size and complexity, everything works.

Weekly Breakdown

  • Week 1: Group up, work on project concept.
  • Week 3: Present design comp.
  • Week 4: First Project Presentations & Critiques.
