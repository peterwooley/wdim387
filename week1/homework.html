WDIM87 Advanced Scripting

Homework: Off The Deep End, due Week 2.

Please turn in your homework by emailing me your GitHub Repository URL and Live URL.

Requirements

  1. Create a new Repository on GitHub.
  2. Get a web host with SSH access and Git installed.
  3. Pick a Server-Side Framework to try (May I suggest CodeIgniter?)
  4. Clone a Working Copy of your Repository onto your Web server.
  5. Download the framework into the Working Copy of your Repository.
  6. Get the framework up and running on your Web server.
  7. Using the framework, create a page that allows me to get the time and date using URLs similar to the following:
    • http://yourwebserver/ajax/date
      (Returns something like {"date": "July 21, 2011, 6:42 pm"})
    • http://yourwebserver/ajax/time
      (Returns something like {"time": 1311273720})
  8. Create a separate page that uses jQuery's $.getJSON function to get the date and and time and show it to the user in some way.
  9. Commit and Push your code changes back to GitHub.
