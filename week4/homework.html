Homework: Final Project

Please turn in your files in a .zip file, using the following naming convention: lastnameFirstname_finalProject.zip

Final Project; due Week 10.

Abstract

You will have six (6) weeks to design, develop, and deploy an Ajax-driven Social Web Application.

You may work on the project alone, in pairs, groups of three, groups of four, or a group of five.

Requirements

  • Your Social Web Application must be something you would find useful.
  • Your SWA must implement the following:
    • Login System with Password Reset
    • Friending/Following (Asymmetric or Symmetric)
    • Voting/Liking/Disliking/Starring
    • Super-User Admin account management with account deletion capability
  • The project must be novel or differentiate itself from its competitors in a positive way.
  • The SWA must utilize Ajax where it benefits the user experience (this includes eye candy).
  • The interaction design must be just that, designed.
  • Generated JavaScript Documentation (using jsdoc-toolkit) must accompany the source code.
  • The application must be hosted on a server that can be accessed from the Internet.

Unrequirements

  • A Secure Login system using SSL to encrypt all sensitive data with grant 10% extra credit.

Ideas

  • Digg clone (but, it would have to rock)
  • Social gaming experience
  • Wiki with inline editing

Grading Rubric

  • 20% Interaction Design
  • 30% Code functions properly, is written clearly, has good comments.
  • 50% Completed the scope of the project, project is of sufficient size and complexity, everything works.

Weekly Project Breakdown

  • Week 4: Group up, develop project concept, & set goals for Week 7 Presentation.
  • Week 5 & 6: Lecture on new concepts.
  • Week 7: Present project iteration progress, set goals for Week 10.
  • Week 8: Progress Critiques with Peter.
  • Week 9: No class.
  • Week 10: Functional Code Complete critiques.
